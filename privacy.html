Privacy Policy

Last updated: June 26, 2025

Data We Collect

To provide a continuous and context-aware conversational experience, we store the following information for each message that triggers the bot:

  • The content of the message that triggered the bot.
  • The response from the bot.
  • The timestamp of the message.
  • The chat ID of the group where the message was sent.

How We Use Your Data

The stored data is used exclusively to:

  • Provide conversational context for the bot to reference in future interactions.
  • Improve the bot's performance and accuracy.
  • Troubleshoot and debug issues.

Data We Do Not Collect

We do not store any personally identifiable information (PII) about our users. We do not collect or store:

  • Usernames or any other personal information from your Telegram account.
  • Messages that do not trigger the bot.

Data Storage and Security

All collected data is securely stored in a private Amazon S3 bucket. We take reasonable measures to protect your data from unauthorized access, use, or disclosure.

Third-Party Services

This bot uses the OpenAI API to generate responses. We send the content of the triggering message to the OpenAI API for processing. We do not have control over the data policies of OpenAI. We recommend you review their privacy policy for more information.

Open Source and Transparency

This project is open-source. You can review the source code on GitHub to verify how we handle your data.

Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page.
